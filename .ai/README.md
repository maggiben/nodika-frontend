# Nordika Frontend AI Framework

This framework is derived from the current repository: a Next.js 16 App Router application with Material UI, React Hook Form, CodeMirror JSON editing, ESLint, Prettier, and Vitest. It is intentionally strict about not inventing API, database, authentication, payment, or deployment conventions that do not exist yet.

- Read `constraints/` before editing.
- Use `harnesses/` to plan and execute changes.
- Use `checks/` before handing work off.
- Update the relevant `skills/` and `docs/` when the architecture materially changes.
