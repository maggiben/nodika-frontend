# Code Review Check

- [ ] Change stays within `src/app` and existing configuration boundaries.
- [ ] Server Component is retained unless browser behavior requires otherwise.
- [ ] Types are explicit; no `any`, ignored errors, or duplicated logic.
- [ ] Material UI components use the shared theme; no Tailwind utilities were introduced.
- [ ] `npm run lint`, `npm run format:check`, `npm test`, and `npm run build` results are recorded.
