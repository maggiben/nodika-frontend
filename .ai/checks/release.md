# Release Check

- [ ] Product title, description, and locale replace the starter metadata in `src/app/layout.tsx`.
- [ ] Starter content/assets/links in `src/app/page.tsx` are removed or intentionally retained.
- [ ] `npm run lint`, `npm run format:check`, `npm test`, and `npm run build` pass.
- [ ] Environment-variable and deployment requirements are documented.
- [ ] Dependency audit findings have been reviewed; no forced remediation was applied blindly.
